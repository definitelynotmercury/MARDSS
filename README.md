# MARDSS
Medical Assistance Request Decision Support System

Developed for the Provincial Social Welfare and Development Office (PSWDO) of Bulacan.

## Tech Stack
- Frontend: React
- Backend: Python Flask
- Database: MySQL
- Analytics: scikit-learn, Pandas

## Modules
- Login / Authentication
- Dashboard
- Analytics
- Forecasting
- Export
